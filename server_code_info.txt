2XX - SUCCESS
3XX - REDIRECT
4XX - Client Error
5XX - Server Error

200 OK: Запрос успешно обработан сервером и вернул запрашиваемые данные.

201 Created: Ресурс был успешно создан на сервере.

204 No Content: Запрос успешно обработан сервером, но в ответе не содержится содержательного содержимого.

400 Bad Request: Сервер не смог понять запрос из-за некорректного синтаксиса или другой ошибки клиента.

401 Unauthorized: Для доступа к ресурсу требуется аутентификация, но она не была предоставлена или недостаточна.

403 Forbidden: Доступ к ресурсу запрещён из-за ограничений, наложенных сервером.

404 Not Found: Запрашиваемый ресурс не найден на сервере.

405 Method Not Allowed: Метод, используемый в запросе (например, GET, POST), не поддерживается для данного ресурса.

500 Internal Server Error: На сервере произошла внутренняя ошибка, которая не позволяет ему обработать запрос.

503 Service Unavailable: Сервер временно не может обрабатывать запросы из-за перегрузки или технических проблем.
